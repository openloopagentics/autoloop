Install the Autoloop reporting skill

Let your agentic loops report their status — project, phases, and commits — to your Autoloop dashboard as they run.

Option 1 — tell Claude Code

Paste this into a Claude Code session:

Install the Autoloop reporting skill from https://daloop-42b47.web.app/skill/install.sh

Claude Code will run the installer below for you.

Option 2 — run it yourself

curl -fsSL https://daloop-42b47.web.app/skill/install.sh | bash

This writes the skill and its bundled CLI to ~/.claude/skills/autoloop-reporting/. Restart Claude Code (or open a new session) — personal skills are discovered at session start.

Want auto-updates? Install as a plugin

The installer above is a one-shot copy. For an updatable install, add the Autoloop plugin marketplace in Claude Code:

/plugin marketplace add openloopagentics/autoloop
/plugin install autoloop-reporting@autoloop

Then enable auto-update under /plugin → Marketplaces → autoloop, or update any time with /plugin marketplace update autoloop.

After installing

# 1. Mint a key in the Autoloop app → API keys
export AUTOLOOP_API_KEY=…

# 2. Point a loop's working dir at your team + project
node ~/.claude/skills/autoloop-reporting/autoloop.mjs init --team <teamId> --project <slug>

From then on the skill auto-activates when a loop should report status. It needs Node 22+ and the AUTOLOOP_API_KEY env var.

Reporting is best-effort — a failed report never blocks the loop. Files: install.sh · SKILL.md · CODEX.md · autoloop.mjs
